Obstacle Avoidance Robot

A classic, effective robotics project built around the Arduino Uno and Sonar technology.

Hardware Interfacing and Logic

The brain of this project is the Arduino Uno. It makes all the decisions based on the information it gets from the sensors. To help the robot "see" its environment, we use an HC-SR04 Sonar Sensor. This sensor works like a bat's echolocation; it sends a sound wave through the Trig pin and listens for the echo through the Echo pin. We connect these pins to standard digital pins on the Arduino. When the echo returns quickly, the Arduino knows an object is too close and it is time to change direction.

For movement, the Arduino alone cannot provide enough power to drive the Two Geared DC Motors. To solve this, we use the L298N Motor Driver. The motor driver connects directly to the battery for high power, while its logic pins (IN1, IN2, IN3, IN4) connect to the Arduino. This allows the Arduino to easily command the motors to go forward, backward, or turn. We also use a small Servo Motor to hold the sonar sensor. The servo is connected to a PWM (Pulse Width Modulation) pin on the Arduino, allowing it to smoothly sweep the sensor left and right to check for the best clear path before moving.

Finally, we need a simple way to control and power the system. A basic Push Button is connected to a digital input pin so the user can easily start or pause the robot's movement. An LED is added as a visual indicator to show when the robot is active or when it detects a sudden block. The entire system is powered by a reliable Rechargeable Li-ion Battery Pack. The battery supplies raw power to the L298N motor driver, and the driver safely steps down the voltage to 5V to power the Arduino Uno and the rest of the logic components.

Arduino Uno Pin Configuration

Here is the straightforward connection guide used to interface all the external components to the main controller.

Component Node Component Pin Arduino Uno Pin
HC-SR04 Sonar Trig Pin Digital Pin 9
HC-SR04 Sonar Echo Pin Digital Pin 10
Servo Motor Signal Wire Digital Pin 11 (PWM)
L298N Motor Driver IN1 (Left Motor) Digital Pin 4
L298N Motor Driver IN2 (Left Motor) Digital Pin 5
L298N Motor Driver IN3 (Right Motor) Digital Pin 6
L298N Motor Driver IN4 (Right Motor) Digital Pin 7
Push Button Output Signal Digital Pin 2
Status LED Anode (+) Digital Pin 13
